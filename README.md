# Nginx 构建脚本

这个代码仓库包含了 Viz 在定制 Nginx 时使用的构建脚本。

## 本体版本

* [PCRE/8.39](http://www.pcre.org/)
* [LibreSSL/2.4.2](http://www.libressl.org/)
* [nginx/1.11.4](http://nginx.org/)

## 附加模块

* [nginx-rtmp-module/1.1.9](https://github.com/arut/nginx-rtmp-module)
* [headers-more-nginx-module/0.31](https://github.com/openresty/headers-more-nginx-module)
* [ngx-fancyindex/0.4.1](https://github.com/aperezdc/ngx-fancyindex)
